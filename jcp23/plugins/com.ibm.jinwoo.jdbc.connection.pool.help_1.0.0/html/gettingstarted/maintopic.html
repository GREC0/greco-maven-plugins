Overview

Welcome to IBM Database Connection Pool Analyzer for IBM WebSphere Application Server.

IBM Database Connection Pool Analyzer helps you to resolve JDBC connection pool related problems through its heuristic analysis engine. You can find more information at http://www.alphaworks.ibm.com/tech/jcp

Features

  • JDBC data source analysis
  • JDBC connection pool configuration analysis
  • JDBC connection chart view
  • Java stack trace view of getConnection method
  • Automatic Detection of Multiple JDBC connections in a single thread
